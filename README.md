# Muhammad Afaq Bhatti — Portfolio Website

A modern, responsive AI/ML Engineer portfolio built with HTML5, Tailwind CSS, and vanilla JavaScript.

## 🚀 Quick Start

1. Open `index.html` in your browser — it works immediately.
2. For local development with live reload, use VS Code + Live Server extension.

## 📁 File Structure

```
portfolio/
├── index.html              ← Main website file
├── README.md               ← This file
├── assets/
│   └── images/
│       ├── pic2026.jpeg    ← Your profile photo (place here)
│       └── quizflow.jpeg   ← QuizFlow project screenshot
├── css/
│   └── style.css           ← All custom styles
└── js/
    ├── projects.js         ← ⭐ ADD NEW PROJECTS HERE
    └── main.js             ← Website logic & interactivity
```

## ➕ Adding a New Project

Open `js/projects.js` and copy the template at the bottom of the file. Fill in your details and save. That's it!

## 🌐 Deploying to GitHub Pages

1. Create a new GitHub repository named `username.github.io` (or any name)
2. Upload all files maintaining the folder structure
3. Go to Settings → Pages → Source: main branch / root
4. Your site is live at `https://username.github.io`

See the PDF User Manual for detailed step-by-step instructions.

## 📧 Contact Form Setup

Replace `YOUR_FORM_ID` in `index.html` with your free Formspree ID from [formspree.io](https://formspree.io).
